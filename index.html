Standard ABM

This example of Agent based modelling was created as part of the practical component of the Programming for Geographical Information Analysts: Core Skills (WUN) course taught by Leeds University as part of the MSc GIS (ODL)

The code creates a set number of sheep (agents) randomly placed onto a "field". The sheep move 1 step at a time in a random direction. After each step, the sheep will eat 10 units of grass before moving on. After eating 100 units, the sheep will be sick. This set of code animates the movements until all the sheep are at least half full (i.e. their stomachs (stores) contains at least 50 units of grass

Enhanced ABM

The enhanced model introduces Wolves to the model. Two wolves, represented as grey diamonds are randomly placed in the field. A wolf can eat a sheep that is with 30 steps of itself. It will then start searching for another sheep from the dead sheeps position. Dead sheep are represented by red X on the map. The model stops running when all the sheep have been eaten.
